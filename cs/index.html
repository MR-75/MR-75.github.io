1.1 Data Representation

Binary and Decimal Prefixes

Denary Prefixes:


  • 1 Byte = \(10^0\) bytes

  • KiloByte = \(10^{3}\) bytes = 1024 bytes

  • MegaByte = \(10^{3}\) Kilobytes = \(10^{6}\) bytes

  • GigaByte = \(10^{3}\) Megabytes

  • Terabyte = \(10^{3}\) Gigabytes


    • Binary Prefixes:
  • 1 Byte = \(2^0\) bytes.

  • KibiByte = \(2^{10}\) bytes = 1024 bytes

  • MebiByte = \(2^{10}\) Kibibytes = \(2^{20}\) bytes

  • GibiByte = \(2^{10}\) Mebibytes

  • Tebibyte = \(2^{10}\) Gibibytes

    • ---

    Number Systems

    Binary
    (we assume that you already know how to convert to and from binary/denary) If a normal binary number system has n bits, there can be $2^n$ possible combination of bits, and as 0 is included, the largest number would be $2^n-1$. ``` 6 bits => 2^6 = 64 combinations largest number = 64-1 = 63 ```
